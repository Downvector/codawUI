UX Design

Project Data Heist
Goal Smartglass is integrated into our core gameplay loop

First steps:

  1. Choose a player-driven genre: stealth games
  2. Avoid time-based challenges for the tablet interactions
  3. User-test our prototype
  4. Be quite sad at the results:

Problem: Players focus on the TV and forget the tablet

Changes:

  1. Realize that adding a tutorial and/or reminders to use the tablet would make for a worse game
  2. Let go of the desire to control when the player looks at the tablet
  3. More flexible systems: players can start focused on the TV for a few minutes, then use the tablet to unlock new areas
  4. "Monitoring gameplay": NPCs can be controlled via a limited number of orders (move, stop, activate skill)
